# 🎨 Telegram Sticker Bot

یک ربات تلگرام پیشرفته برای ساخت استیکر متنی با پشتیبانی کامل از زبان فارسی

## ✨ ویژگی‌ها

- 🎨 ساخت استیکر متنی با فونت‌های فارسی و انگلیسی
- 📦 ساخت پک استیکر جدید
- ➕ اضافه کردن استیکر به پک موجود
- 🌐 پشتیبانی از فرمت WebP بدون خطا
- 🔧 رابط کاربری ساده و کاربرپسند
- 📱 مدیریت همزمان چندین کاربر
- 🎯 ذخیره خودکار وضعیت کاربران

## 🚀 دیپلوی در Vercel

1. این ریپازیتوری را فورک کنید
2. در Vercel پروژه جدید بسازید و به گیت‌هاب متصل کنید
3. متغیرهای محیطی را تنظیم کنید:
   - `BOT_TOKEN`: توکن ربات تلگرام
   - `VERCEL_URL`: آدرس پروژه Vercel

## 📋 متغیرهای محیطی

```env
BOT_TOKEN=your_telegram_bot_token_here
VERCEL_URL=your_vercel_app_url_here
PYTHON_VERSION=3.11
```

## 🛠️ تکنولوژی‌ها

- Python 3.11
- python-telegram-bot 20.7
- Pillow 10.3.0
- Flask 2.3.3
- arabic-reshaper
- python-bidi

## 📦 ساختار پروژه

```
mybot/
├── api/
│   ├── __init__.py
│   └── index.py          # کد اصلی ربات
├── fonts/                # فونت‌های فارسی و انگلیسی
├── templates/            # قالب‌های پس‌زمینه استیکر
├── requirements.txt      # کتابخانه‌های مورد نیاز
├── vercel.json          # تنظیمات دیپلوی Vercel
└── runtime.txt          # نسخه پایتون
```

## 🤖 دستورات ربات

- `/start` - شروع و منوی اصلی
- `/newpack` - ساخت پک استیکر جدید
- `/addtopack` - اضافه کردن به پک موجود
- `/help` - راهنمای ربات

## 🔧 راهنمای استفاده

1. ربات را با `/start` شروع کنید
2. گزینه ساخت استیکر جدید را انتخاب کنید
3. متن مورد نظر خود را ارسال کنید
4. استیکر ساخته شده را به پک اضافه کنید
